# 🐍 Python Learning Manager

## 🚀 项目简介

本仓库用于记录个人在 Python 与 AI 开发方向的学习过程，包含从基础语法到简单项目实战的完整学习路径。

当前内容主要包括：

- 📘 Python 基础语法学习
- 🧩 函数与模块化编程
- 🏗️ 类与对象（OOP）
- 📂 文件与 JSON 操作
- 🌐 Streamlit Web 开发
- 🤖 DeepSeek / OpenAI API 调用
- 📊 CSV 数据处理
- 🕷️ 网络机器人基础
